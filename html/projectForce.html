ESXi 3.5 Server from spare computer parts


Hardware

Dell OptiPlex GX270

  • Pentium 4 3.00 GHz
  • 4 GB RAM
  • 2 x Intel Gigabit Ethernet(82540EM, 82541PI)
  • 1GB USB Hard Drive
  • 500 GB IDE 7200 rpm/8MB cache

Software

  • ESXi 3.5

Virtualizing is the 'big-trend' for computing and datacenters in 2010. Corporate infrustructures are being converted to virtual machines to reduce overhead costs such as electricity, physical space, and hardware. Project Force allows me to run 5 virtual machines from just one hardware resource.


USB Hard Drive storing ESXi.


A "Frontside"(LAN facing) interface and "Backside" interface(via crossover cable) allows most efficient network traffic.


Mos Eisley runs Cacti providing quick history and status graphs for all hosts using SNMP.
